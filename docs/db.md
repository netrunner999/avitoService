# Таблицы в базе данных приложения

## Таблица users - пользователи

Поля:
id - uuid пользователя (ПК)
login - логин
password - пароль
email - почта
phone - телефон
is_active - активность пользователя (флаг)
created_at - время и дата регистрации пользователя
updated_at - время и дата последнего обновления пользователя

## Таблица sessions - сессии

Поля:
sessions_id - идентификатор сессии (ПК)
user_id - uuid пользователя из users (ВК)
created_at - время и дата создания сессии
expires_at - время и дата окончания сессии

## Талица adverts - объявления

Поля:
id - uuid объявления (ПК)
user_id - uuid пользователя из users (ВК)
status - статус объявления (active/sold/cancelled)
created_at - время и дата создания объявления
completed_at - время и дата продажи
updated_at - время и дата обновления объявления
